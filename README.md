# Student Registration API

A simple REST API for student registration built with Express.js or Flask.

## Features

- Register students
- View all registered students
- JSON request/response handling
- Input validation

---

## Required Fields

- name
- email
- course

---

## API Endpoints

### Register Student

POST /api/students

### Get All Students

GET /api/students

---

## Run Express Version

npm install
node server.js

---

## Run Flask Version

pip install flask
python app.py
